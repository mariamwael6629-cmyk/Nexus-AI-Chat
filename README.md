# Nexus AI — Intelligence That Evolves With You 🧠✨

**Nexus AI** is a modern, memory-first AI platform built for knowledge workers, developers, and founders. Unlike traditional chatbots that forget context between sessions, Nexus securely retains facts, tech stacks, and user preferences across all past conversations to deliver deeply personalized, highly contextual intelligence.

---

## 🚀 Key Features

*   **🧠 Persistent Memory Engine:** Retains user facts, preferences, and workspace context forever. Reference something said months ago, and Nexus will seamlessly integrate it into the current thread.
*   **🔍 Interactive Memory Center:** A transparent, comprehensive dashboard to browse, search, edit, or delete anything the AI has learned. Memories are categorized by *Preferences, Work & Skills, Projects, Profile, and Interests*.
*   **📊 Productivity Analytics Dashboard:** Provides a bird's-eye view of your data with statistics on total conversations, saved memories, message counts, weekly activity graphs, and specific model usage distributions (*Nexus Pro, Nexus Fast, Nexus Reason*).
*   **💼 Built-in AI Power Tools:** 
    *   **Code Interpreter:** Run and test code blocks in real-time.
    *   **Document Intelligence:** Upload, read, and analyze PDFs, spreadsheets, and images.
    *   **Web Search:** Live access to current internet data.
    *   **Image Gen:** Quick visual creation inside the workspace.
*   **🎨 Tunable AI Personality:** Customize response attributes ranging from concise and highly technical to exploratory and warm.

---

## 🛠️ Tech Stack

This project is built using modern, industry-standard web technologies:

*   **Frontend:** React.js, TypeScript, Tailwind CSS (Custom Dark-Mode Architecture)
*   **Backend:** Node.js, Express
*   **Database & Auth:** Supabase / PostgreSQL
*   **AI Integration:** Large Language Models (LLMs) with custom vector-embeddings for persistent long-term memory retrieval.
